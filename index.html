Claude Usage Float

拡張機能のサポート / Privacy Policy ページ

Claude Usage Float は、Claude の使用量とレート制限状況をコンパクトなポップアップで確認できる ブラウザ拡張です。

プライバシーポリシー サポートへ連絡

主な機能

  • 5時間 / 7日ウィンドウの使用量表示
  • 設定可能な自動更新
  • ローカルキャッシュのみを利用

English Summary

Claude Usage Float is a compact browser extension for checking Claude usage and rate-limit status from your existing Claude session.
