Claude Usage Float Extension

データ収集について

この拡張機能は、個人情報を第三者サービスへ収集・送信しません。

動作内容

  1. ブラウザ内の Claude.ai の sessionKey cookie を読み取ります。
  2. そのセッションを使って Claude の公式 organization / usage API を呼び出します。
  3. organization ID、使用量データ、設定をブラウザ内ストレージに保存します。

保存されるデータ

  • sessionKey: chrome.storage.session に5分間キャッシュされます。
  • Usage data: chrome.storage.local に保存されます。
  • Settings: ポーリング間隔を chrome.storage.local に保存します。

送信先

  • リクエスト送信先は https://claude.ai/api/organizations* のみです。
  • アナリティクス、テレメトリ、第三者トラッキングは使用しません。
  • データを販売したり、無関係な第三者へ共有したりしません。

使用する権限

  • alarms: 定期ポーリングのスケジュールに使用します。
  • storage: 使用量データと設定をブラウザ内に保存します。
  • cookies: Claude.ai のセッション cookie を読み取ります。
  • host_permissions: https://claude.ai/*: Claude の公式 API を呼び出します。

アクセスできないもの

  • パスワード
  • 会話履歴全体
  • 他のウェブサイトや他の拡張機能のデータ

データ保持

データはブラウザ内にのみ保存され、拡張機能のアンインストール、ブラウザストレージの削除、 または Claude.ai からのログアウト時に消去されます。

ポリシー変更

データ取り扱いに変更がある場合は、このページおよびソースコードに反映します。

Privacy Policy (English Summary)

This extension does not collect or transmit personal data to unrelated third parties.

  • It reads the sessionKey cookie from Claude.ai in your browser.
  • It calls Claude's official organization and usage endpoints.
  • It stores usage data and settings locally in browser storage.
  • No analytics, telemetry, or third-party tracking is used.
